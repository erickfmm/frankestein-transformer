\label{sec:annex-summary-tables}

This appendix provides a complete reference of all components implemented in the Frankestein Transformer codebase: 35 sequence mixers across six families, 23 optimizers, 6 normalization variants, 43 activation functions, and 4 embedding methods. Each table is self-contained and cross-references the bibliography.
