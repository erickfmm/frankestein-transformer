\section{Tablas de Resumen Completas}
\label{sec:annex-summary-tables}

Este anexo proporciona una referencia completa de todos los componentes implementados en el código base de Frankestein Transformer: 35 mezcladores de secuencia en seis familias, 23 optimizadores, 5 variantes de normalización, 43 funciones de activación y 4 métodos de embedding. Cada tabla es autocontenida y referencia la bibliografía.
