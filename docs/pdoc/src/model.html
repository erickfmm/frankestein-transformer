609class TormentedBertFrankenstein(nn.Module):
610    """TORMENTED-BERT-Frankenstein: Hybrid mixed-architecture Transformer encoder.
611
612    This is the flagship model. It stacks ``num_layers`` :class:`HybridLayer`
613    blocks, each configured by ``layer_pattern``, and repeats the entire
614    stack ``num_loops`` times (looped depth). The logical depth is
615    ``num_layers * num_loops``.
616
617    Key architectural features:
618
619    * **17+ attention mixer families** dispatched per-layer via
620      :class:`HybridLayer`.
621    * **Looped depth**: the physical layer stack is iterated ``num_loops``
622      times, sharing parameters across loops for parameter-efficient deep
623      computation.
624    * **Mixture-of-Experts (MoE) FFN**: per-token top-k expert routing with
625      weighted expert outputs.
626    * **BitNet b1.58**: ternary weight quantization via :class:`BitLinear`
627      when ``use_bitnet`` is True.
628    * **Factorized embeddings**: reduced-dimension embedding lookup +
629      projection via :class:`FactorizedEmbedding`.
630    * **Mixture-of-Depths**: per-layer token routing where only the
631      top-capacity tokens are updated; auxiliary load-balancing loss is
632      accumulated and exposed via ``last_auxiliary_losses``.
633    * **Normalization**: ``layer_norm``, ``dynamic_tanh`` (DyT), or
634      ``derf`` (Dynamic Erf).
635    * **Positional encoding**: RoPE or HoPE, applied inside attention
636      mixers.
637
638    Attributes:
639        config: The :class:`UltraConfig` used to build the model.
640        emb: Token embedding layer (:class:`FactorizedEmbedding` or
641            ``nn.Embedding``).
642        dropout: Embedding dropout layer.
643        layers: ModuleList of ``num_layers`` :class:`HybridLayer` blocks.
644        final_norm: Final normalization before the output head.
645        head: Output projection to vocabulary logits (Linear or BitLinear).
646        last_auxiliary_losses: Dict of auxiliary losses from the most recent
647            forward pass (e.g. ``"mixture_of_depths_router_loss"``).
648        last_mixture_of_depths_stats: Dict of MoD statistics from the most
649            recent forward pass (``"average_selected_fraction"``,
650            ``"raw_router_aux_loss"``).
651    """
652
653    def __init__(self, config):
654        """Build the Frankenstein encoder from an UltraConfig.
655
656        Args:
657            config: :class:`UltraConfig` instance with all model
658                hyperparameters.
659        """
660        super().__init__()
661        self.config = config
662        self.last_auxiliary_losses = {}
663        self.last_mixture_of_depths_stats = {}
664
665        if config.use_factorized_embedding:
666            self.emb = FactorizedEmbedding(config)
667        else:
668            self.emb = nn.Embedding(config.vocab_size, config.hidden_size)
669        self.dropout = nn.Dropout(config.dropout)
670
671        self.layers = nn.ModuleList(
672            [
673                HybridLayer(config, layer_type=config.layer_pattern[i % len(config.layer_pattern)])
674                for i in range(config.num_layers)
675            ]
676        )
677
678        self.final_norm = get_norm(config)
679        proj_cls = BitLinear if config.use_bitnet else nn.Linear
680        self.head = proj_cls(config.hidden_size, config.vocab_size)
681
682    def forward(self, input_ids):
683        """Run the full looped-depth encoder forward pass.
684
685        Iterates the physical layer stack ``num_loops`` times, tracking a
686        global ``logical_layer_idx``. Accumulates Mixture-of-Depths auxiliary
687        losses across all layers and stores them in
688        ``last_auxiliary_losses``.
689
690        Args:
691            input_ids: Integer token indices of shape ``(B, S)``.
692
693        Returns:
694            Logits tensor of shape ``(B, S, vocab_size)``.
695        """
696        x = self.emb(input_ids)
697        x = self.dropout(x)
698
699        logical_layer_idx = 0
700        mixture_of_depths_aux_losses = []
701        mixture_of_depths_selected_fractions = []
702        for _ in range(self.config.num_loops):
703            for layer in self.layers:
704                x = layer(x, logical_layer_idx=logical_layer_idx, input_ids=input_ids)
705                if layer.use_mixture_of_depths and layer.last_mixture_of_depths_aux_loss is not None:
706                    mixture_of_depths_aux_losses.append(layer.last_mixture_of_depths_aux_loss)
707                    mixture_of_depths_selected_fractions.append(
708                        layer.last_mixture_of_depths_selected_fraction
709                    )
710                logical_layer_idx += 1
711
712        x = self.final_norm(x)
713        if mixture_of_depths_aux_losses:
714            raw_aux_loss = torch.stack(mixture_of_depths_aux_losses).mean()
715            weighted_aux_loss = (
716                raw_aux_loss * float(self.config.mixture_of_depths_router_aux_loss_weight)
717            )
718            self.last_auxiliary_losses = {
719                "mixture_of_depths_router_loss": weighted_aux_loss,
720            }
721            self.last_mixture_of_depths_stats = {
722                "average_selected_fraction": sum(mixture_of_depths_selected_fractions)
723                / len(mixture_of_depths_selected_fractions),
724                "raw_router_aux_loss": float(raw_aux_loss.detach().item()),
725            }
726        else:
727            self.last_auxiliary_losses = {}
728            self.last_mixture_of_depths_stats = {}
729        return self.head(x)

TORMENTED-BERT-Frankenstein: Hybrid mixed-architecture Transformer encoder.

This is the flagship model. It stacks num_layers HybridLayer blocks, each configured by layer_pattern, and repeats the entire stack num_loops times (looped depth). The logical depth is num_layers * num_loops.

Key architectural features:

  • 17+ attention mixer families dispatched per-layer via HybridLayer.
  • Looped depth: the physical layer stack is iterated num_loops times, sharing parameters across loops for parameter-efficient deep computation.
  • Mixture-of-Experts (MoE) FFN: per-token top-k expert routing with weighted expert outputs.
  • BitNet b1.58: ternary weight quantization via BitLinear when use_bitnet is True.
  • Factorized embeddings: reduced-dimension embedding lookup + projection via FactorizedEmbedding.
  • Mixture-of-Depths: per-layer token routing where only the top-capacity tokens are updated; auxiliary load-balancing loss is accumulated and exposed via last_auxiliary_losses.
  • Normalization: layer_norm, dynamic_tanh (DyT), or derf (Dynamic Erf).
  • Positional encoding: RoPE or HoPE, applied inside attention mixers.
Attributes:
  • config: The UltraConfig used to build the model.
  • emb: Token embedding layer (FactorizedEmbedding or nn.Embedding).
  • dropout: Embedding dropout layer.
  • layers: ModuleList of num_layers HybridLayer blocks.
  • final_norm: Final normalization before the output head.
  • head: Output projection to vocabulary logits (Linear or BitLinear).
  • last_auxiliary_losses: Dict of auxiliary losses from the most recent forward pass (e.g. "mixture_of_depths_router_loss").
  • last_mixture_of_depths_stats: Dict of MoD statistics from the most recent forward pass ("average_selected_fraction", "raw_router_aux_loss").

 85@dataclass
 86class UltraConfig:
 87    """Single-source-of-truth configuration for all Frankenstein model variants.
 88
 89    Every hyperparameter lives here. The schema is validated in
 90    ``__post_init__`` and enforced by ``configs/schema.yaml`` for YAML-based
 91    training configs.
 92
 93    Attributes:
 94        vocab_size: Vocabulary size for token embeddings. Default: 50000.
 95        hidden_size: Dimensionality of hidden states throughout the model.
 96            Must be divisible by ``num_heads``. Default: 2048.
 97        num_layers: Number of physical :class:`HybridLayer` blocks in the
 98            stack. Default: 12.
 99        num_loops: Number of times the layer stack is repeated (looped depth).
100            Logical depth = ``num_layers * num_loops``. Default: 2.
101        layer_pattern: Ordered list of mixer types assigned to each physical
102            layer. The pattern is cycled modulo its length when
103            ``num_layers`` exceeds the pattern length. Supported values:
104            ``"ode"``, ``"retnet"``, ``"retnet_attn"``, ``"titan_attn"``,
105            ``"standard_attn"``, ``"sigmoid_attn"``, ``"mamba"``,
106            ``"sparse_transformer_attn"``, ``"longformer_attn"``,
107            ``"bigbird_attn"``, ``"sparsek_attn"``, ``"nsa_attn"``,
108            ``"sparge_attn"``, ``"fasa_attn"``, ``"gla_attn"``,
109            ``"deltanet_attn"``, ``"gated_deltanet_attn"``,
110            ``"hgrn2_attn"``, ``"fox_attn"``, ``"gated_softmax_attn"``,
111            ``"engram_attn"``. Default: ``["retnet", "ode", "mamba",
112            "titan_attn"] * 3``.
113        ode_solver: ODE solver for ``ode`` mixer layers. One of ``"rk4"``
114            (Runge-Kutta 4th order) or ``"euler"``. Default: ``"rk4"``.
115        ode_steps: Number of ODE integration steps per ``ode`` layer.
116            Default: 2.
117        retention_heads: Number of retention heads for ``retnet`` /
118            ``retnet_attn`` layers. Default: 8.
119        num_heads: Number of attention heads for standard / sparse / gated
120            attention mixers. Default: 16.
121        num_experts: Total number of FFN experts when ``use_moe`` is True.
122            Default: 8.
123        top_k_experts: Number of experts activated per token in MoE routing.
124            Default: 2.
125        dropout: Dropout probability applied after embeddings and within
126            attention layers. Default: 0.1.
127        use_bitnet: If True, replace all ``nn.Linear`` layers with
128            :class:`BitLinear` (ternary weight quantization, BitNet b1.58).
129            Default: True.
130        norm_type: Normalization layer type. One of ``"layer_norm"``,
131            ``"dynamic_tanh"`` (DyT), or ``"derf"`` (Dynamic Erf).
132            ``"rms_norm"`` is NOT valid. Default: ``"dynamic_tanh"``.
133        use_factorized_embedding: If True, use :class:`FactorizedEmbedding`
134            with reduced embedding dimension + projection. Default: False.
135        factorized_embedding_dim: Embedding dimension when factorization is
136            enabled. Default: 128.
137        use_embedding_conv: If True, apply a Conv1d over the factorized
138            embedding stream before projection. Default: True.
139        hope_base: Base frequency for HoPE (Hybrid Positional Encoding).
140            Default: 10000.0.
141        hope_damping: Damping factor for HoPE high-frequency components.
142            Default: 0.01.
143        rope_base: Base frequency for RoPE (Rotary Position Embedding).
144            Default: 10000.0.
145        rope_scaling: Scaling factor applied to RoPE frequencies.
146            Default: 1.0.
147        use_hope: Legacy toggle for HoPE. Automatically aligned with
148            ``positional_encoding`` in ``__post_init__``. Default: True.
149        positional_encoding: Explicit positional encoding scheme. One of
150            ``"hope"`` or ``"rope"``. If None, inferred from ``use_hope``.
151            Default: None.
152        use_moe: If True, replace the dense FFN with a Mixture-of-Experts
153            FFN block. Default: True.
154        use_mixture_of_depths: If True, apply per-layer token routing where
155            only the top-capacity tokens are updated; remaining tokens are
156            passed through unchanged. Default: False.
157        mixture_of_depths_capacity_ratio: Fraction of tokens selected per
158            layer when Mixture-of-Depths is active. Must be in (0, 1].
159            Default: 0.5.

Single-source-of-truth configuration for all Frankenstein model variants.

Every hyperparameter lives here. The schema is validated in __post_init__ and enforced by configs/schema.yaml for YAML-based training configs.

Attributes:
  • vocab_size: Vocabulary size for token embeddings. Default: 50000.
  • hidden_size: Dimensionality of hidden states throughout the model. Must be divisible by num_heads. Default: 2048.
  • num_layers: Number of physical HybridLayer blocks in the stack. Default: 12.
  • num_loops: Number of times the layer stack is repeated (looped depth). Logical depth = num_layers * num_loops. Default: 2.
  • layer_pattern: Ordered list of mixer types assigned to each physical layer. The pattern is cycled modulo its length when num_layers exceeds the pattern length. Supported values: "ode", "retnet", "retnet_attn", "titan_attn", "standard_attn", "sigmoid_attn", "mamba", "sparse_transformer_attn", "longformer_attn", "bigbird_attn", "sparsek_attn", "nsa_attn", "sparge_attn", "fasa_attn", "gla_attn", "deltanet_attn", "gated_deltanet_attn", "hgrn2_attn", "fox_attn", "gated_softmax_attn", "engram_attn". Default: ["retnet", "ode", "mamba", "titan_attn"] * 3.
  • ode_solver: ODE solver for ode mixer layers. One of "rk4" (Runge-Kutta 4th order) or "euler". Default: "rk4".
  • ode_steps: Number of ODE integration steps per ode layer. Default: 2.
  • retention_heads: Number of retention heads for retnet / retnet_attn layers. Default: 8.
  • num_heads: Number of attention heads for standard / sparse / gated attention mixers. Default: 16.
  • num_experts: Total number of FFN experts when use_moe is True. Default: 8.
  • top_k_experts: Number of experts activated per token in MoE routing. Default: 2.
  • dropout: Dropout probability applied after embeddings and within attention layers. Default: 0.1.
  • use_bitnet: If True, replace all nn.Linear layers with BitLinear (ternary weight quantization, BitNet b1.58). Default: True.
  • norm_type: Normalization layer type. One of "layer_norm", "dynamic_tanh" (DyT), or "derf" (Dynamic Erf). "rms_norm" is NOT valid. Default: "dynamic_tanh".
  • use_factorized_embedding: If True, use FactorizedEmbedding with reduced embedding dimension + projection. Default: False.
  • factorized_embedding_dim: Embedding dimension when factorization is enabled. Default: 128.
  • use_embedding_conv: If True, apply a Conv1d over the factorized embedding stream before projection. Default: True.
  • hope_base: Base frequency for HoPE (Hybrid Positional Encoding). Default: 10000.0.
  • hope_damping: Damping factor for HoPE high-frequency components. Default: 0.01.
  • rope_base: Base frequency for RoPE (Rotary Position Embedding). Default: 10000.0.
  • rope_scaling: Scaling factor applied to RoPE frequencies. Default: 1.0.
  • use_hope: Legacy toggle for HoPE. Automatically aligned with positional_encoding in __post_init__. Default: True.
  • positional_encoding: Explicit positional encoding scheme. One of "hope" or "rope". If None, inferred from use_hope. Default: None.
  • use_moe: If True, replace the dense FFN with a Mixture-of-Experts FFN block. Default: True.
  • use_mixture_of_depths: If True, apply per-layer token routing where only the top-capacity tokens are updated; remaining tokens are passed through unchanged. Default: False.
  • mixture_of_depths_capacity_ratio: Fraction of tokens selected per layer when Mixture-of-Depths is active. Must be in (0, 1]. Default: 0.5.
  • mixture_of_depths_router_aux_loss_weight: Weight for the auxiliary load-balancing loss in Mixture-of-Depths routing. Must be >= 0. Default: 0.0.
  • ffn_hidden_size: Hidden size of the FFN intermediate layer. If None, defaults to hidden_size * 2. Default: None.
  • ffn_activation: FFN activation function. One of "silu" (SiLU / Swish) or "gelu" (GELU). Default: "silu".
  • embedding_conv_kernel: Kernel size for the embedding Conv1d when use_embedding_conv is True. Default: 3.
  • mode: Model mode. "encoder" for bidirectional (MLM) or "decoder" for autoregressive causal generation. The model_class=frankesteindecoder preset forces mode=decoder at runtime. Default: "encoder".
  • engram_max_ngram_size: Highest N-gram order for Engram memory layers (range 2..max). Default: 3.
  • engram_n_heads_per_ngram: Number of hash heads per N-gram order in Engram layers. Default: 4.
  • engram_embed_dim_per_head: Embedding dimension per Engram hash head. Default: 32.
  • engram_kernel_size: ShortConv kernel width for Engram layers. Default: 4.
  • engram_seed: RNG seed for Engram hash multipliers. Default: 42.
Raises:
